## Prerequisites

- [Node.js](https://nodejs.org/en/);
- [npm](https://www.npmjs.com);
- [Vue CLI](https://cli.vuejs.org/);
- [Git](https://git-scm.com/);

## Getting started
```shell
git clone https://github.com/mattiapontonio/asteroids.git
cd asteroids
npm i
npm run build
export PORT=<port>
export API_KEY=<key>
npm start
```

## Lifecycle scripts
```shell
npm run
```

## Install
```shell
npm install
```
